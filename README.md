# Sudoku-Master-C-
A high-performance Sudoku game featuring a recursive backtracking solver, a randomized puzzle generator, and a modular Object-Oriented design.

## Key Features
* **Efficient Backtracking Algorithm:** Capable of solving any valid Sudoku puzzle using optimized recursion.
* **Randomized Puzzle Generation:** Create unique boards on the fly with adjustable difficulty levels.
* **Modular Architecture:** Strict separation of concerns to ensure maintainability:
    * **Backend:** Logic-heavy 0-indexed coordinate system.
    * **Frontend:** User-facing API using 1-indexed coordinates for intuitive interaction.
